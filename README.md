# AgenciaTurismoconTours2

Proyecto desarrollado para la asignatura Desarrollo Orientado a Objetos I.

## Funcionalidades implementadas
- Programación Orientada a Objetos.
- Encapsulamiento.
- Herencia.
- Composición.
- Manejo de excepciones personalizadas.
- Lectura de datos desde archivo de texto (`tours.txt`).
- Almacenamiento de información mediante `ArrayList`.
- Búsqueda de tours por nombre.
- Filtrado de tours por tipo.
- Gestión de servicios turísticos mediante herencia.
- Visualización de rutas gastronómicas.
- Visualización de paseos lacustres.
- Visualización de excursiones culturales.
